Before contacting support

  • Install the latest version and test again.
  • Include your Windows version and what happened step by step.
  • Add a screenshot or short video when possible.
  • Mention whether the issue is always reproducible or random.

Quick FAQ

Does Munjez require an internet connection?

No. Munjez is designed to be offline-first and works locally on your device.

How can I suggest a new feature?

Open a GitHub issue and describe the feature, why it helps, and how you expect it to work.

How long does a response usually take?

Response times vary, but tickets with clear reproduction steps and media are handled faster.

Why does Windows show a security warning when I install Munjez?

Munjez is not yet signed with a paid code-signing certificate, so Windows SmartScreen flags it as an unrecognized app. This is normal for independent apps and does not mean the app is harmful. To proceed, click More info then Run anyway. If the file is fully blocked, right-click it → Properties → check Unblock → OK. Full step-by-step instructions are on the installation guide page.
